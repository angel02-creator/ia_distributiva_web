# IA Distributiva Visual

Esta demo es un sitio web responsivo sobre inteligencia artificial distributiva. Está diseñado para verse bien en móviles, tablets y pantallas grandes, y usa imágenes SVG locales para ofrecer una experiencia visual limpia.

## Archivos principales

- `index.html` — página principal
- `styles.css` — estilos responsivos y visuales
- `script.js` — menú móvil interactivo
- `assets/images/*.svg` — imágenes seguras y locales

## Cómo usar

1. Abre la carpeta `site`.
2. Haz doble clic en `index.html`.
3. Usa cualquier navegador moderno (Chrome, Edge, Firefox, Safari).

No necesitas descargar ninguna aplicación adicional para ejecutar esta demo.
